## 项目介绍
本项目基于百度智能云 AI 开放平台接口，使用 Python + Flask 搭建 Web 平台，实现三大 AI 功能：
1. OCR 文字识别（识别图片中的文字）
2. 图像物体识别（识别图片中的物体类别）
3. 文字转语音合成（输入文字自动朗读播放）

前端实现图片预览、结果格式化展示，界面简洁可用，适合课程作业展示。

## 项目功能
✅ 页面展示学号 + 姓名
✅ 图片 OCR 文字识别
✅ 图像高级物体检测识别
✅ 文本语音合成与自动播放
✅ 图片上传预览
✅ 结果 JSON 格式化展示
✅ 文件格式与大小校验

## 技术栈
- 后端：Python + Flask
- AI 接口：百度智能云 OCR / 图像识别 / 语音合成
- 前端：HTML + JavaScript
- 语音播放：pygame
